I just released The Stellar Expedition v1.2.0 with some handy updates. The game now shows your trajectory in real time, so you know exactly where your spacecraft will land—making it much more playable than before, when only I could handle it.

Trajectories can still be slightly off in rare edge cases. On very low-end mobiles, FPS can drop with long trajectories, though that’s uncommon. To improve performance, I plan to refactor the code so past and future trajectory predictions run simultaneously instead of sequentially—cutting processing roughly in half. Honestly, I have no idea why I didn’t think of that sooner!

Data handling is more reliable too. The game now detects corrupted files and tries to fix them before launch; if repair fails, it resets the file so the game still loads instead of crashing.

Also, the game now has all levels unlocked, so you won’t get stuck or have to spend time completing everything just to see the full game. Completed levels show a small check mark, so you still get that sense of achievement. Go ahead and play the game by clicking the link below.
